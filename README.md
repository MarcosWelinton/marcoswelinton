### 🚀 Sobre Mim & Meus Objetivos

Sou estudante do **3º semestre de Sistemas de Informação**, focado em me tornar um Desenvolvedor de Software completo. 

Minha abordagem de desenvolvimento une três pilares essenciais: a robustez lógica de algoritmos eficientes, o gerenciamento estruturado de dados via **SQL** e a sensibilidade de design proporcionada por estudos de **UI/UX**. Como entusiasta de hardware, busco sempre entender o impacto do código diretamente na arquitetura física do computador para garantir a máxima performance dos sistemas.
